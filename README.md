# Element Plot Workbench

一个在浏览器中运行的地球化学散点图与元素配分图工作台。

## 功能

- 读取 Excel、CSV 和多个工作表
- 自由选择散点图横纵坐标
- 按 `Mission + Sub_Type` 分层配色
- 使用 `Ti_Type` 区分点形状
- 使用 `Ti_Type → Mission → Sub_Type` 三级树筛选
- 折线图可任意勾选元素并调整横轴顺序
- 新增任意两列的比值列
- 绘制单样品元素配分图
- 按分类字段计算并绘制平均配分线
- 导出 2400 × 1800 PNG 或可编辑 SVG
- 图片按“横坐标名 vs 纵坐标名”自动命名
- 图例位于黑色作图区外，元素刻度自动倾斜以避免重叠
- 可按 Ti_Type、Mission 或具体 Sub_Type 自定义填充色、边框色和形状

所有数据均在浏览器本地处理，不会上传到服务器。
