# Element Plot Workbench

一个在浏览器中运行的地球化学散点图与元素配分图工作台。

## 功能

- 读取 Excel、CSV 和多个工作表
- 自由选择散点图横纵坐标
- 按 `Mission + Sub_Type` 分层配色
- 使用 `Ti_Type` 区分点形状
- 使用 `Ti → Mission → Sub` 三级树筛选，同时兼容旧字段名 `Ti_Type/Sub_Type`
- 折线图可任意勾选元素并调整横轴顺序
- 新增任意两列的比值列
- 绘制单样品元素配分图
- 按分类字段计算并绘制平均配分线
- 导出 2400 × 1800 PNG 或可编辑 SVG
- 图片按“横坐标名 vs 纵坐标名”自动命名
- 图例位于黑色作图区外，元素刻度自动倾斜以避免重叠
- 可按 Ti、Mission 或具体 Sub 自定义填充色、边框色和形状
- 提供完整可编辑的 Type 样式表和 Ti → Mission → Sub 层级样式表
- 可一键选择工作表中的全部 `_N` 标准化元素列
- 可将全部图例样式导出为 JSON，并在其他数据文件中重新导入
- 内置用户确认的图例样式配置作为默认值
- 控制栏宽度可调，数据、坐标、分类、计算和导出面板可折叠
- 操作面板按数据、计算、作图、分类、图例和导出的工作顺序排列
- 分类筛选与图例样式为独立折叠面板
- 图中图例可直接拖动，并通过右下角控制柄缩放
- 元素配分图序列可在当前浏览器会话中命名保存、恢复和删除
- 图表英文使用 Calibri，中文使用仿宋回退字体
- 坐标刻度朝内，可手动指定最小值、最大值和主刻度间隔
- 分类平均折线可显示平均值 ±1 SD 阴影

所有数据均在浏览器本地处理，不会上传到服务器。
